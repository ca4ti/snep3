Alias de expresiones regulares - Agregar / Editar

Alias expresión regular se utiliza para facilitar la creación de reglas de negocio que comparten las mismas expresiones regulares.
Para registrar un nuevo alias acaba de elegir un nombre y determinar una o más expresiones regulares que se utilizarán para comparar fuentes y destinos en las reglas de negocio.
Para el alias es lo suficientemente válida que al menos una de las expresiones casarse con el parámetro especificado en la norma.
