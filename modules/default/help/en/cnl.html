Update CNL

The update the CNL allows that is updated the database tables containing the names of cities, DDD and prefixes phones of all telephone operators.
This data is used to display the reports on which city the call originated or was intended.
IMPORTANT:
1) The data provided by ANATEL (www.anatel.gov.br)
2) The update file is prepared by the Opens Technology and is in JSON format.
3) There are two types of files: for fixed and mobile telephony and their names are in the following format
* Mobile: cnl-movel-dd-mm-aaaa.json
* Fixed.: cnl-fixo-dd-mm-aaaa.json
where: dd = day, mm = month, yyyy = year => date the file was generated.
