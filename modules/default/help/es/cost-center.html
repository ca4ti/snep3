Centro de Costo - Añadir / Cambiar

Esta rutina permite la inclusión / edición de un centro de coste.
El Centro de costos son utilizados por Asterisk para establecer el 'accountcode' de cada conexión de salida, registrar esta información en la tabla cdr.

Un centro de costos consta de los siguientes campos:
Código: El código puede ser de 1, 3 o 5 dígitos, siempre NÚMERO, y distribuidos en un máximo de 3 'casas', que determinan su herarquia. Definición:
9 - código de un dígito, se considera el código Padre. Debe haber sólo dos (2) códigos de PAI en el sistema - una para los insumos y una para SALIDAS.
9.99 - código de tres dígitos, es considerado el hijo de código de primera generación
9.99.99 - código de cinco dígitos, es considerado el hijo de código de segunda generación
Ejemplo:
1 - Conexiones de Entrada
1.01 - Departamento Comercial
1.01.01 - Director Comercial
1.01.02 - Vendedores
1.01.03 - Telemarketing
Nombre: Define un nombre para el centro de coste;
Descripción: Breve descripción de la finalidad del centro de coste
Tipo: Define qué tipo de conexión se aplicará el Centro de Costo: entrada o salida

Observación: No puede editar el código del centro de coste.

Centro de Costo - CSV Exportación

Esta rutina salidas del centro de costos registradas de sistema en un archivo .csv.
