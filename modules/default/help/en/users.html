Usuários - Incluir / Alterar

Essa rotina é usada para criação/edição de um usuário de acesso ao sistema.

Nome: Nome do usuário.
Senha: Senha para acesso ao sistema
Email: E-mail do contato. Caso a senha seja perdida é possível a recuperação por e-mail.*
Perfil: Perfil no qual o usuário fará parte.
*Caso o usuário esqueça sua senha, é possível recupera-lá por e-mail através do campo "Esqueceu a senha" na tela de login. Por default é criado um usuário admin onde o mesmo possui permissões a todas rotinas do Snep. Ao recuperar a senha, será enviado uma mensagem ao e-mail cadastrado informando um código aleatório com validade de 1 hora. No snep será solicitado o código e nova senha.
IMPORTANTE: O Snep deve possuir algum serviço de e-mail. Ex: Postfix

Usuários - Permissões

O usuário herda todas permissões do perfil vinculado a ele. Mas é possível alterar as permissões diferenciando o usuário do perfil.
Por exemplo: O perfil A que possui 3 usuários vinculados a ele e possui permissão de leitura e escrita de centro de custos. Na opção permissões do usuário é possível acrescentar permissões ou retirar uma destas duas permissões. Onde todos os usuários do perfil terão as permissões escolhidas no perfil e somente este usuário possuirá permissões diferentes do grupo.

Usuários - Exportar CSV

Esta rotina exporta dados de todos os usuários cadastrados no arquivo formado .csv.
