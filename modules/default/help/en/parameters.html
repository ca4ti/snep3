Configurações - Parâmetros

Esta rotina permite que algumas configurações de usabilidade do SNEP sejam definidas ou ajustadas pelo usuário.

Nome da Empresa: Define o nome da Empresa onde o sistema está instalado.
Hostname (*): Define um número IP ou Hostname do servidor onde o Asterisk está instalado.
Usuário (*): Define o nome do usuário que será usado para conectar-se com o Asterisk.
Senha (*): Define a senha que será usada para conectar-se com o Asterisk.
(*) A ligação do SNEP, através destes campos se dá com o Asterisk através do arquivo /etc/asterisk/manager.conf
Aplicação de Conferência: Define as configurações da sala de conferência.

Configuração de Localização
Fuso Horário: Define o fuso horário que o snep tomará de base para suas aplicações.
Localidade do Sistema: Define o país onde o snep esta instalado.
Linguagem do Sistema: Define o idioma da interface.

Configuração da Gravação de Ligação
Aplicação da Gravação: Define as configurações da usadas para as gravações.
Parâmetros de Gravação: Opção usada pelo aplicaivo de gravação.

Configuração do Ramal
Range de Ramais/Agentes: Define a margem de ramais/agentes que serão cadastrados por ex: 2000-2020 - Serão aceitos ramais/agentes apenas dos numeros entre essa faixa, 20 ramais/agentes 2000 ao 2020, abaixo ou acima não serão aceitos.
