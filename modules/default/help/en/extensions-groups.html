Extension Group - Add / Change

This routine is used to create / edit Extension group in Snep.
Groups can be of type user, administrator or no. This type indicates heritage.
For example, if a business rule is created allowing connections ara Management and the Commercial group is of type Administrator, all extensions that pertentecem the Commercial group will also be considered by rule as administrators.

Selecting Extensions : It is possible, to create / edit a group, you select the extensions that will be part of the same.
Available extensions : Contains the list of extensions available in the system. Each extension possesses the current group to which it belongs in brackets: Ex: 1001 (admin)
