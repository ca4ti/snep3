Extensions - Add / Edit

This routine allows the registration / edit an extension in the system.
The registered extensions should be in the range of variation of marge extensions configured on the system parameters.

Extension: Defines the code that identifies the extension.
Extension Group: Sets the group of extensions which the station belongs.
Pickpup Group: Capture group that the extension will be part. Extensions in the same pickup group allowed us to capture liga√ßos each other.

INTERFACE TECHNOLOGY
Technology: Defines which channel technology the extension will use to make / receive calls.
SIP/IAX: Channels with SIP or IAX technology.
Khomp: Technology extensions Khomp.
Virtual: Extensions with virtual interfaces are mapped in a trunk. When selecting a trunk interface to the virtual extension, the link to it will be made by dialing through the trunk to the extension number.
Manual: Allows you to specify which channel / technology will be utlizada in creating the extension.

OPTIONS SIP / IAX
Limit of simultaneous calls: Sets the maximum number of simultaneous calls to this extension.
NAT: Sets the branch connection will or not through a NAT.
Quality control: Sets whether or not quality control on the links Extension.
Direct media: Sets passage or not the RTP packets by Asterisk.
DTMF Mode: Sets the type of frequency tones used in dialing.
CODECS: Define the sequence of CODECS to be preferred for Extension.

OPTIONS KHOMP
Board: Defines which of the boards in the extension system will be part.
Channel: Defines which channel of the selected card.

OPTION VIRTUAL
Trunk: Sets the trunk to be mapped.

OPTION MANUAL
Manual: Sets the channel / technology to be used.

ADVANCED
Voicemail: Sets whether or not the branch will use Voice Mail.
E-mail: Defines an e-mail Extension.
Block: Lock for the extension. Whether to allow outgoing calls.
Enable follow me: Allows the extension to make follow me.
Minutes control: Defines whether it will be stipulated time to the links that extension or not.
Full Time: Total time that the extension can be in connection.
Control Type: Sets whether the minutes are controlled by year, month or day.

Extensions - Export CSV

This routine export data of registered extensions to a .csv file.

Extensions - Add extensions

This routine includes a range of extensions. The range can be separated by - (hyphen) or (semicolon).
Example ranges:
1000-1002 -> The system will register the extensions 1000,1001 and 1002.
1000;1002 -> The system will register the extensions 1000 and 1002.
