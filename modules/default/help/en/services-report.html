Relatório serviços utilizados

Permite que você visualize e gere relatórios de serviços utilizados por determinado ramal, lista de ramais ou grupos de ramais.

Período: Define o intervalo de datas/hora do qual se deseja extrair as informações.
Ramais: Informa quais ramais estão envolvidos com o evento.
Serviço habilitado: Relaciona qual serviço esta habilitado para o ramal.
Status do serviço: Ultimo registro do ramal.
Exportar para CSV: Gera relatório no formato .csv.
