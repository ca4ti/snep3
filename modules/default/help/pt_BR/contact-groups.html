Grupo de Contatos - Incluir

Esta rotina cadastra um novo grupo de contatos. Basta fornecer um nome para o grupo e opcionalmente você pode selecionar os contatos listados para facilitar a administração.

Grupo de Contatos - Alterar

Esta rotina edita um grupo existente. Poderá haver edição dos nomes e membros do grupo.

Grupo de Contatos - Excluir / Migrar

Essa função é realizada quando há exclusão do grupo de contatos. A função realiza migração dos contatos associados de um grupo para um novo grupo ou efetua a exclusão dos contatos associados.
