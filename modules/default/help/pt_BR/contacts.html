Contatos - Cadastrar / Editar

Preencha o formulário corretamente com as informações do contato.Os campos nome e telefone são de preenchimento obrigatório.
Código: Identificará o cadastro do Contato. Este campo o Sistema preenche um código automáticamente.
Nome: Nome identificador do contato.
Grupo: Grupo de contato no qual ele vai pertencer.
Endereço, Cidade, Estado CEP: Dados do endereço do contato.
Telefone: Identifica o telefone do Contato. É possível cadastrar vários números ao usuário.

Contatos - Importar CSV

Esta rotina permite que a partir de um arquivo no formato .csv, uma lista de contatos possa ser importada para o sistema. Após selecionar o arquivo e clicar em “Enviar”, uma tela com os campos detectados no arquivo .csv será apresentada. É necessário relacionar cada campo com o referido campo do cadastro do contato. A primeira linha é a chamada “linha de cabeçalho”, que identifica cada coluna do arquivo e as demais são as linhas de dados. A vírgula separa cada campo e mesmo que não exista dados para ele, o mesmo deve estar presente.
Exemplo: nome,cep,cidade,estado,telefone
Pedro da Silva,80123333,Curitiba,PR,041991122234
Carlos Fagundes,,Criciuma,SC,04878900012

Contatos - Export CSV

Esta rotina exporta os contatos para um arquivo .csv. É possível exporta todos os contatos ou especificamente de um determinado grupo.

Contatos - Remover multiplos

Esta rotina exclui todos os contatos do grupo escolhido.
