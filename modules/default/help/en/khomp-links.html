Status Links Khomp

This system routine shows only information about the status of the links Khomp plates. If no Khomp card is identified by the system, a warning message is displayed.
Shows information of Khomp cards installed in the system.

There are basically 4 types of Khomp plates: E1, FXS, FXO and GSM. The system identifies the plates only for its "position in the system", always preceded by the letter B.
Select the board: allows you to select on which cards we want to see the available information.
Type Report:
Analytical –> shows the information of the links and each channel of each plate ;
Synthetic –> shows information of the links and a summary of channels of each plate;
View status of the channel: Allows you to display or not the status of each channel (active, in use, etc ...)
