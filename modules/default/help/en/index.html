System

This routine displays overall information system performance.

System Information: Exibe informações sobre o sistema operacional, banco de dados, distribuição Linux, versão do Kernel, tipo de CPU, etc.
Modulles: Exibe informações sobre os módulos do SNEP e suas respectivas versões.
Estatística do PBX: Exibe informações sobre quantidades de ramais, troncos e rotas cadastradas no Snep
Uso da memória: Exibe informações sobre uso da memória RAM física, lógica, de cache e de troca (swap)
Uso do disco: Exibe informações sobre o uso do disco de cada partição do sistema
