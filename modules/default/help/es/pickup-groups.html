Grupo de captura - Incluir / Alterar

Essa rotina é usada para criação/edição do grupo de captura no Snep. Os Grupos de Captura fazem parte do Cadastro de um Ramal e ajudam a determinar o Ramais que poderão capturar ligações entre si.

Seleção de Ramais: É possivel, ao criar um grupo, que você selecione os ramais que farão parte do mesmo.
