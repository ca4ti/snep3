Extensión Grupo - Añadir / Cambiar

Esta rutina se utiliza para crear y edición de grupo extensión en Snep.
Los grupos pueden ser de tipo de usuario, administrador o no. Este tipo indica patrimonio.
Por ejemplo, si se crea una regla de negocio que permite conexiones para la Gestión y el grupo comercial es de tipo Administrador, todo las extensiones pertentecem el grupo comercial también será considerado por regla como administradores.

Selección de Extensiones : Es posible crear / editar un grupo, seleccionar las extensiones que formarán parte de la misma.
Extensiones disponibles : Contiene la lista de extensiones disponibles en el sistema. Cada extensión posee el grupo actual a la que pertenece en soportes. Ejemplo: 1001 (admin)
