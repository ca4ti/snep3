Contatos - Añadir / Editar

Rellene el formulario correctamente con la información de contacto. Los campos nombre y teléfono se requieren.

Código: Identificar el registro del contacto. El sistema se llena un código automáticamente.
Nombre: Contacto nombre identificador.
Grupo: Contactar grupo pertenecerá.
Dirección, Ciudad, Estado y Código Postal: Los datos de la dirección de contacto.
Teléfono: Identifica el teléfono de contacto. Se pueden registrar varios números de usuario.

Contactos - CSV Importació n

Esta rutina permite a partir de un archivo en formato .csv, una lista de contactos se pueden importar en el sistema. Después de seleccionar el archivo y haga clic en "Enviar", se mostrará una pantalla con los campos detectados en el archivo .csv. Es necesario relacionar cada campo con dicho campo de contacto maestro. La primera línea se llama la "línea de cabecera" que identifica a cada columna del archivo y los otros son las líneas de datos. La coma separa cada campo e incluso si hay datos a la misma, debe estar presente.
Ejemplo: nombre,código postal,ciudad,estado,teléfono
Pedro da Silva,80123333,Curitiba,PR,041991122234
Carlos Fagundes,,Criciuma,SC,04878900012

Contactos - CSV Exportación

Esta rutina exportar contactos a un archivo .csv. Puede exportar todos los contactos o específicamente para un grupo particular.

Contactos - Quitar múltiple

Esta rutina elimina todos los contactos del grupo seleccionado.

Contactos - Acceso directo en los enlaces

El Snep tiene una característica enlaces a contactos. Al marcar * 12 + código de contacto se realiza la conexión con el primer número vinculado a este contacto.
Por ejemplo: Hay un contacto A, que tiene el código 1 y tiene en los teléfonos de archivos 4899999999 y 4899888888. Al marcar el teléfono * 121 se realiza la conexión con el número 4899999999.
