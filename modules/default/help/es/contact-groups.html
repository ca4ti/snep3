Grupo Contactos - Añadir

Esta rutina registra un nuevo grupo de contactos. Simplemente proporcione un nombre para el grupo y, opcionalmente, puede seleccionar los contactos de la lista para facilitar la administración.

Grupo Contactos - Cambiar

Esta edición de rutina un grupo existente. Se pueden editar los nombres y los miembros del grupo.

Grupo Contactos - Borrar / Migrar

Esta función se realiza cuando hay exclusión del grupo de contacto. A função realiza migração dos contatos associados de um grupo para um novo grupo ou efetua a exclusão dos contatos associados.
