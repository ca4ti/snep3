Group Contacts - Add

This routine registers a new contact group. Just provide a name for the group and optionally you can select the contacts listed for ease of administration.

Group Contacts - Edit

This routine edit an existing group. There may be editing the names and group members.

Group Contacts - Delete / Migrate

This function is performed when there is exclusion from the contact group. The function performs migration of contacts associated to a group to a new group or makes the exclusion of contacts associated.
